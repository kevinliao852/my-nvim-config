# my-nvim-config

[![Review style](https://github.com/kevinliao852/my-nvim-config/actions/workflows/style-check.yaml/badge.svg)](https://github.com/kevinliao852/my-nvim-config/actions/workflows/style-check.yaml)

A modern Neovim configuration built as an AGENT-ready IDE for AI-assisted development.
It is designed to support a structured workflow across ACP, MCP, and local AI tools,
so editing, navigation, context handoff, and automation stay in one place.

## What it covers

- AGENT-friendly editor workflows for focused, tool-driven development
- ACP and MCP integration patterns for external context and model orchestration
- AI-assisted editing, refactoring, and code navigation
- A clean Neovim setup tuned for day-to-day development
